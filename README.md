# 🚀 AI Portfolio Builder

An AI-powered web application that allows users to create, customize, and publish professional developer portfolios in minutes.

---

## ✨ Features

- 🎨 Multiple portfolio templates
- 📝 User-friendly form to enter details
- 🤖 AI-powered content enhancement (coming soon)
- 🔗 Live portfolio preview
- 🌐 Shareable portfolio links
- 📄 Resume download support (planned)

---

## 🛠️ Tech Stack

- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- AI Integration: OpenAI API

---

## 📌 Project Goal

To simplify portfolio creation for students and developers by automating design, content writing, and deployment using AI.
